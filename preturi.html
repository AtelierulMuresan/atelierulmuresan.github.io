PRETURI

Care este pretul serviciilor noastre?

SCANARE 3D
Pretul pentru o sedinta de scanare este de 50 RON iar durata unei sesiuni este de maxim o ora.

PRINTARE 3D
In cazul printarii 3D un pret aproximativ este de 2 RON / gram material consumat pentru imprimantele FDM si 3 RON / ml material consumat in cazul imprimantei DLP

GRAVURA / PRELUCRARE LASER
Pentru gravare / prelucrari laser pretul este 2 RON / minut

PRELUCRARE CNC
Pentru prelucrari CNC pretul este 15 RON / minut

Preturile pot varia in functie de complexitatea modelului / operatiunilor.Pentru o estimare cat mai buna si o oferta de pret cat mai corecta este nevoie de un MODEL 3D in format STL (in cazul printarii 3D sau prelucrarii CNC) sau de un fisier DXF (in cazul gravurii laser sau prelucrarii CNC) Daca nu aveti un model 3D sau un desen 2D va putem ajuta si cu servicii de design specializat, contra cost. Costul acestuia fiind negociat in functie de nevoile clientului
